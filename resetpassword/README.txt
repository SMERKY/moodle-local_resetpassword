ABOUT

This plugin adds a page for an external service to post to, when posted to with a valid email address, and the correct hashed private key, the moodle function to send a password reset email will be called.

INSTALLATION

Install this plugin in your /local folder
Then login as an admin and go to
Site Administration -> Notifications

This should trigger the installation process


CONFIGURATION

To configure this plugin go to:
Site Administration -> Plugins -> Local Plugins -> Password Reset
